## バージョン管理ルール

- プロンプトを変更する際は必ずファイル内の`バージョン`と`更新日`を更新する
- コミットメッセージは `feat:` `fix:` `docs:` のプレフィックスを使う
- 大きな変更は `v1.0.0 → v2.0.0`、小さな修正は `v1.0.0 → v1.1.0`
